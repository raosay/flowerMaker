# 一枝春 — 聊赠春意（PWA）

在线花艺搭配应用。品牌名取自南朝陆凯《赠范晔》：

> 折花逢驿使，寄与陇头人。  
> 江南无所有，**聊赠一枝春**。

以「春」喻花色与心意，不直说花；以「赠」点明送人的产品意图。

灵感来源：[flority.digital](https://flority.digital/) · [Alexandra A](https://www.instagram.com/alexandra.aurielle/)

## 快速开始

```bash
npm install
npm run dev
```

```bash
npm run build
npm run preview
```

## 功能

- 50 种花材，画布自由摆放
- 分享向导：花艺墙、致/来自、留言、PNG / 系统分享
- 草稿自动保存、PWA 可安装
